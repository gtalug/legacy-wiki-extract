Introduction

Templates can be added freely, but this page is more for official templates. Templates are generally locked because they're fairly essential and can be very annoying if a malicious user mucks with them. See wiki policies for more.

Creating a template

Create a template the same way you would create a page. Make a link with {{template}} and follow it to edit it. Then you can add text to it. Then, any use of {{template}} will instead display its text.

To insert the content of the template permanently into the page, do a {{subst:template}}. When saved, the page will absorb the contents of that template.

Editing a template

When editing a page, you may notice a chunk of it is only a template. To edit that section of text, look to the very bottom of the page (not the editor window, the page) and you'll notice a list of links to the templates being used in the page being edited. Follow the appropriate template link and edit the template instead.

Please note that some templates may still be editable only by an administrator.. write a note to one of the wiki maintainers for help.

Wiki Badges

A wiki badge is a marker. This marker is set on a page so that it can be easily found later, using backlinks.

  • Do (backlinks)
  • Fixme (backlinks)
  • Port (backlinks) -- Pages to be imported. Give it a URL to the original site.
  • Presenter (backlinks) -- Presenters.

Use a wiki badge like so:

* I want to remind myself to come back to this note. {{Do|Do this later}}

Appears as:

  • I want to remind myself to come back to this note. ~Do: Do this later

Now to follow up with a badge, follow its "backlinks". This will give you a search page which lists all the pages which have that badge in them. You can visit the badge (e.g. Template:Do) and then select "What links here" in the sidebar, or use a fancy link such as Special:Whatlinkshere/Template:Do.

Footnotes

Fn and Fnb are used together:

Example:

This is some text that has a footnote.{{fn|1}} {{fnb|1}} my footnote

appears as:

This is some text that has a footnote.[1]

[1]: my footnote
