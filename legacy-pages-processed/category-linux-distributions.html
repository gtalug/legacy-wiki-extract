Linux Distributions

There are numerous "Distributions" of Linux, that is, collections of software that normally include:

  • A bootstrap process (typically LILO or GRUB) that is just smart enough to load and run a Linux Kernel
  • A Linux_Kernel
  • A body of software normally including
    • Init, the program that runs atop the kernel that starts up all the other services
    • Bash or other shells to run "glue" code
    • The whole host of applications and services that people associate with a "Linux system"
