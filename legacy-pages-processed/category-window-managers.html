Window Managers

A window manager is computer software that controls the placement and appearance of windows within the [X Window System]. This typically includes such things as:

  • Determining the positions of windows
  • Drawing decorations on the outside of windows
  • Responding to controls (often using mouse or keyboard) to modify the locations and sizes of windows
  • Determining which window should receive keyboard input

Some window managers go further, adding additional sorts of functionality:

  • Virtual screen management - which establishes "virtual screens" to allow you to have many non-overlapping windows at one time
  • Icons or docks to indicate hidden/minimized windows
  • Menu systems to permit running applications
  • Panel systems to display system information or to "hold" menus, docks, screen views, and such
  • Some window managers have further functionality to interact with other applications

Note that all of these functions are distinct from the notion of a Category:Desktop Environment
