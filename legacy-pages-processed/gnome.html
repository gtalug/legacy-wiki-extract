GNOME

GNOME is a free software project aiming to create a powerful Desktop Environment (as distinct from window manager).

Distinctive include:

  • Libraries are implemented in C
  • The framework was intended to be language agnostic, allowing apps to be written in many languages
  • Applications tend to start with the letter "G"
