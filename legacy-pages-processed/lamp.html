Culture > LAMP 

acronym: Linux Apache MySQL PHP

The "P" can stand for any of the (fairly numerous) languages that start with the letter "P", including Perl, Python, PHP.

Wikipedia:LAMP


LAMP is the most famous use, although variations exist:

  • LAMPP -- Linux, Apache, MySQL, PHP/Perl
  • LAPP -- Linux, Apache, PostgreSQL, (some language starting with "P")
  • LAMP -- Linux, Apache, Middleware, PostgreSQL (where "Middleware" can be any sort of glue/scripting language)
