LedgerSMB

LedgerSMB is a community-developed accounting system written in Perl that uses PostgreSQL for data storage. It began as a fork from the SQL-Ledger system written by Dieter Simader, which was duel-licensed under the GPL and a traditional proprietary license. Note that "SMB" is short for "Small and Medium Business." LedgerSMB offers, as advantages over the software it forked from:

  • Enhanced security
  • Better Windows support
  • More reports
  • An Open Development Model
  • Better data integrity controls
