Subject: Linux Users Group in Toronto, Canada
Reply to Author | Forward | Print | Individual Message | Show original | Report Abuse

Unlike many large cities in the U.S., Toronto still does not
have a Linux Users Group (LUG), even though I believe there
is tremendous interest in such an organization.

Have you ever wondered:

        -- how to put your PC on the Internet
        -- how to build a shared library
        -- how to configure X Windows
        -- how to debug your programs using GDB
        -- how to get your exotic hardware to run under Linux
        -- how to use Linux in your business

Would you like to meet key Linux developers such as:

        -- Linus Torvalds   (kernel developer)
        -- Eric Youngdale   (DLL shared libraries, cluster patches)
        -- H. Lu            (Linux GCC and libc guru)
        -- Alan Cox         (net3 networking)
        -- Orest Zborowski  (X port, virtual consoles)

Would you like to find out about new Linux projects such as:

        -- wt              3D Virtual reality engine
        -- WINE            Windows emulator
        -- dosemu          DOS emulator
        -- IBCS            environment for running binaries from other Unices

If you would like to help in creating the Toronto LUG, please send
e-mail to L...@light-house.gts.org. If there is a good response,
I will set up a mailing list for the organizers.

 Regards,

    Laszlo
