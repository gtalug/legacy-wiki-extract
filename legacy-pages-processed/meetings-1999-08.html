Using Tcl/Tk to construct a Graphical User Interface. He'll do an overview, the big picture, which would be most useful for someone who has a problem to solve and isn't sure if Tcl is the tool to use. He'll also provide some pointers to information, and describe the structure of a Tcl program in this application, which includes serial communications with hardware. And include the 'tour des widgets' to show the nifty gadgets that are available for GUI building.
