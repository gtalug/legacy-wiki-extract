Web Application Programming with XML.

Presented by David S. Arnold.

TLUG meetings are held at UofT on the second Tuesday of each month at 7:30 pm.

Date
Tuesday November 14th, 2000
Time
7:30 pm
Place
LM-158 Lash Miller Chemical Laboratories, University of Toronto 80 St. George Street, Room 158 (St. George St. & Willcocks St.) (see map link below for detailed directions)
Topic
Web Application Programming with XML.
Speaker
David S. Arnold, programmer, Starnix, Inc. http://www.starnix.com
Description
An introduction to the use of Extensible Markup Language and related technologies (XSL, DTDs) in a web application programming environment (examples with Java and Perl). Focus on free software packages such as Apache, expat, Cocoon.
