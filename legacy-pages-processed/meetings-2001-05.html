The e-smith server and gateway.

Presented by Kirrily Robert.

TLUG meetings are held at UofT on the second Tuesday of each month at 7:30 pm.


Date
Tuesday May 8, 2001
Time
7:30 pm


Topic
The e-smith server and gateway.
Speaker
Kirrily "Skud" Robert
Description
e-smith is an open source Internet server and gateway solution for small to medium businesses or home networks. It combines the ease-of-use of a network appliance with an open, extensible architecture. This presentation will outline the features of the system and describe the modular approach taken to encourage Linux developers to use e-smith as a base for their own software products or services.
