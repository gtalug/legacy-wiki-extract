Topic

Shell Scripting Recipes: A Problem-Solution Approach, by Chris F.A. Johnson, published May 2005 by Apress

Speaker

Chris F.A. Johnson

Description

Shell Scripting Recipes (ISBN 1590594711) contains more than 200 scripts and functions that can be used on any Linux system without any modification. It is intended for all users who want to get more out of their computers. The book also demonstrates how the shell (on Linux, normally this is Bash) can be used as an all-purpose programming language.

I can bring a few copies of the book for those who wish to purchase them. The price is $35.00 each. Please e-mail me (cfaj at freeshell.org) if you would like a copy.
