Topic

An Open Source Geographic Information System - GRASS

Speaker

Doc Robinson

Description

This presentation will provide an overview of the open source geographic information system GRASS. Examples of its use on Linux/Unix systems will be presented. Its linkage to the open source Wikipedia:R (programming language) statistical analysis project will also be shown.
