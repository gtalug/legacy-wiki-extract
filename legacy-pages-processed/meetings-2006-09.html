Topic

Performance and Capacity Planning

Speaker

David Collier-Brown <davecb@spamcop.net> or <davecb@datacenterworks.com>

Description

Much tuning and all capacity planning deals with resource bottlenecks, because actual performance used to be hard to measure. This is no longer true. We can now do application performance measurement with a packet sniffer and a spreadsheet. This is a talk on how it's done and what we can do with the results.

The slides are in , and it and the spreadsheet are available on request.

Speaker Bio

Dave is an ex-Sun performance engineer and capacity planner, now playing consultant as part of Data Center Works.
