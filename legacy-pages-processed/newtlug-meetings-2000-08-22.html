Announcement

Date: Tuesday, August 22 Time: 6:30pm Place: SGI, 5800 Explorer Drive, Mississauga (see http://toronto.linux.ca/newtlug1.html for directions and Map)

Topic: Samba - Linux and Windows interconnectablity Presenter: Terry Tanski

This will be a howto and demonstration style talk:

  • demos of both the client side and server side including connecting to/from Win2000, Win98
  • using SWAT - "Samba Web Administration Tool" web based configuration interface
  • encrypted password/authentication
  • sample scheme(s) to backup entire (mixed Linux and Windows) network to one or more Linux Samba servers

For those who would like to readup on this topic here are a couple of resources: 

   http://us1.samba.org/samba/samba.html 
   http://www.redhat.com/support/docs/tips/Samba-Tips/Samba-Tips.html
