Topic:

The Seaside Web-Application Framework

Seaside is a highly productive Smalltalk environment for building and deploying highly interactive web-applications. The talk will explain Seaside's key characteristics, which are: html generation framework, call/answer, and components. These features combine to abstract away the mundane details of the HTTP protocol, so the developer can concentrate on the business function.

We will go through the simple steps required to create a new application, illustrate how the debugging process is simple and interactive. We will look at some sample applications which connect to a PostgreSQL database (or maybe we can try to develop one on-the-fly).


Presenter:

Yanni Chiu <yanni@rogers.com> Yanni has over 15 years of industry experience, including such things as C and Fortran compiler work, desktop publishing, UNIX system administration, and implementing systems in financial, manufacturing, and publishing industries. For many years now, he has been a Smalltalk, Java, and web-application consultant.
