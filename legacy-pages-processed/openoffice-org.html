OpenOffice.org

Originally a proprietary system called StarOffice, OpenOffice.org is now an Open Source project managed by Sun Microsystems. It provides a quite credible "office suite" that is largely compatible with some of the major components of Microsoft Office. It uses an open data format called OpenDocument.

It consists of various components:

  • Writer - a word processing module similar to Microsoft Word
  • Calc - a spreadsheet module similar to Lotus 123 or Microsoft Excel
  • Impress - a presentation program similar to PowerPoint
  • Draw - a vector graphics drawing program similar to CorelDraw!
  • Base - a database application somewhat similar to Microsoft Access
