SAMBA

Wikipedia:Samba (software) is a network filesystem compatible with Microsoft's CIFS system. As of version 3, Samba not only provides file and print services for various Microsoft Windows clients but can also integrate with a Windows Server domain, either as a Primary Domain Controller (PDC) or as a Domain Member.
