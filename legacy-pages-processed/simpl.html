SIMPL - Synchronous Interprocess Messaging Project for LINUX

Send/Receive/Reply messaging , first popularized in commercial RTOS's such as Wikipedia:QNX, is:

  • intuitive and simple to understand (think function call)
  • an immensely powerful way of designing distributed software applications

This project aims to bring this powerful messaging paradigm to the Open Source application developer.
