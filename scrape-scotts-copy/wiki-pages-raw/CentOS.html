CentOS is a community-supported alternative to RHEL.

Note that CentOS is a distribution that uses RPM for package management.

Centos user comments

[edit]

Leah

I have started to use this distribution with some of my clients who desire something like RHEL but are not intested in paying for support and "official" maintenance. It is an interesting idea. I am not overly fond of RHEL to begin with, but I like the function that it provides for users.
[edit]

Sim

Like Leah, I have just started with this distro. ver 4 installed like a dream - a nice feature was the ability to set up raid disks with LVM on top. Not being very skilled, I'm having a hang of a time getting Samba shares to be seen by M$ OS's - testparm did not show any errors. CentOS complies with SELinux (Security Enhanced) so, I'm wondering whether some port is closed by default... Other than that, I pretty much installed all the software I could and everything works just fine: good server software like GNOME, Evolution, Firefox etc. etc. etc. RH Up2date makes keeping current very easy. It comes with YUM too - but I've had some config issues... I've also used webmin to goof around with various configs and create and drop users etc. My next project will be to install Oracle on it to see how it serves out data.
