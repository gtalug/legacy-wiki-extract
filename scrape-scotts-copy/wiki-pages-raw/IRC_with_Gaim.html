Introduction

This tutorial will show you how to join our IRC channel, #toronto-lug on irc.freenode.net.

For demonstration purposes:

  • Username: JohnDoe75
  • Password: asdf12345

If you haven't already installed Gaim please install it, then continue.

[edit]

Use Gaim to create an IRC account

  1. Select Tools Â» Accounts
  2. Click Add
  3. Drop-down select Protocol: IRC
  4. Type the screen name you wish to use as Screen Name: JohnDoe75
    • (You can easily change this later if someone else has this screenname.
  5. Ensure that the Server: irc.freenode.net is specified.
  6. The Password: must be left blank, as irc.feenode.net is a public server.
  7. Click Save
    • Alias: and other options can be left blank for now.)
  8. In the Accounts window Select JohnDoe75@irc.freenode.net and check the checkbox for Online

Once you have connected you can begin to participate, however, we recommend that you register your nickname.

NOTE: If you get a prompt reading NickServ: (notice) This nickname is owned by someone else and you have not yet registered, you will need to choose a different nickname (screen name).

[edit]

Join the #toronto-lug IRC channel

  1. Select Buddies Â» Add Group... and make the name of the group IRC. Click ADD.
  2. Check Buddies Â» Show Empty Groups to make the IRC group you just created visible.
  3. Alt-click on the IRC group select Add Chat
  4. Select Account: JohnDoe75@irc.freenode.net (IRC) .
  5. In the first textbox type in Channel: #toronto-lug.
  6. Select Group: IRC
  7. Click Add
  8. Alt-click on #toronto-lug in your buddy list and select Join

You're now in #toronto-lug, hang out and talk to us! Or leave a message, if we're not around. As we gain members in different time zones, we hope to eventually have the channel active 24/7!

Register your Nickname

If your identity is important to you or to others (partiularly if your nick has had special privileges given to it in certain chatrooms) you should register it (absolutely vital if you are admin or operator of a room).

For the purposes of this demonstration we'll prentend that the password is asdf12345.

NOTE: The following instructions are specific to irc.freenode.net and other servers using the same software.

  1. Select Buddies Â» Add Group... and make the name of the group IRC. Click ADD.
  2. Check Buddies Â» Show Empty Groups to make the IRC group you just created visible.
  3. Alt-click the new IRC group on your buddy list and select Add a Buddy.
  4. In the first textbox type in Screen Name: NickServ.
  5. Select Group: IRC
  6. Select Account: JohnDoe75@irc.freenode.net (IRC) at the bottom of the window that appears.
  7. Click Add
  8. Alt-click NickServ and select IM
  9. Send the message REGISTER asdf12345
    • You can also do this by sending /msg NickServ REGISTER asdf12345 from any chatroom.
  10. Now identify yourself with the message IDENTIFY asdf12345.
    • You can also do this by sending /msg NickServ IDENTIFY asdf12345 from any chatroom.

NOTE: For a list of generic commands send the message /help from any chatroom.

NOTE: For a list of NickServ commands send the message HELP to it.

WARNING: All of this information should only go to "NickServ", you should never be sending it to any IRC channel or any individual who is not NickServ.

Automate your IRC experience

[edit]

Autojoin the #toronto-lug channel on login

Alt-click on #toronto-lug in your buddy list and select Auto-join if you want to join the #toronto-lug IRC channel automatically every time you sign on to Gaim (most people will want to do this).

[edit]

IDENTIFY automacitally NickServ on each login

So you've registered your name and you don't want to have to type "identify [password]" every time you log in? You can either use the GAIM IRC Helper plugin (for the more technically inclined user) or you can set a buddy pounce, which is a kinda like a beefy version of AIM's "alert me when user comes online" feature.

  1. Select Tools Â» Buddy Pounce Â» Set Buddy Pounce.
  2. Select JohnDoe75@irc.freenode.net (IRC).
  3. Type Buddy Name: NickServ.
  4. Select Pounce When: Sign on.
  5. Check Pounce Action: Send a message IDENTIFY asdf12345.
    • (You may with to uncheck all other options)
  6. Check Save this pounce after activation.
  7. Click Save.
