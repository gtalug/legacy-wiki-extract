MAME

MAME (Multiple Arcade Machine Emulator) is an open source software arcade machine emulator that allows software from old video arcade machines to be run. A current generation multi-gigahertz PC has no trouble emulating hardware used in anything through 1990s video games. MAME is available for a number of platforms, everything from MS-DOS to current Linux distributions. MAME supports over 8,000 different coin operated video games. MythTV uses MAME to provide video game functionality to a PVR box.

Linux supports several editions of MAME. The most commonly used edition is SDLMame, which has superseded the abandoned XMame (MAME for X Windows) project.

MAME itself is a command line application. There are a number of GUI "front ends" to provide a convenient way to access games. These include GNOME video arcade and the Mythgame front-end for MythTV. These front-ends do not come with MAME: they must be installed as separate applications.

General information on MAME can be found at Wikipedia's MAME Page.


[edit]

Installation and Required Files

MAME and usable MAME front-ends are provided with many Linux distributions. Most (or all) of these are open sourced: if they are not provided with your distro, you can build the software from the source files.

The MAME software emulates the coin operated game hardware such as the CPU, sound cards and video cards. MAME knows how to run games but does not come with any data files such as the game ROM's (that is, the game program itself), sound samples, compressed hard drives (CHD's) or icons, cabinet art, flyers, marquees and screenshots for the front-ends. You will need to acquire the necessary data files to actually play games. MAME is very version-sensitive: you'll need the version of these files to match the particular version of MAME you are using. Newer games, particularly ones with CHD's, can require a lot of space but many older games only require the correct ROM's to run. Due to legal reasons, GTALUG cannot direct you to these data files, but they are readily available on the Internet. However, sources of free, openly legal ROMS are listed in the Externals Links section below.

Once you have acquired the data files you need, edit the mame.ini configuration file to refer to the directories where the data files are stored. You can verify the configuration using "mame -showconfig" at the command line.

Cabinets

Some old style video game fans have built cabinets made to look like the old video game cabinets, right down to the coin slots in the front. There are several sets of plans for how to build an arcade style cabinet available on the Internet. If you are comfortable using a woodworking router and doing a bit of soldering you can have an arcade style game in your recreation room.

There are a number of plans available on the Internet for building your own cabinet.

When shopping for coin acceptors, keep in mind that coin acceptors can be built sensitive enough to accept US quarters and reject Canadian quarters. So, when shopping for coin acceptors pay attention to which currency, or currencies the acceptor will handle.

[edit]

Legalities

The MAME software is legal, it just emulates old arcade machine hardware. Further a small number of copyright holders have given conditional permission for their games be copied (the most common condition being the games not be used in a for-profit enterprise). For the vast majority of cases, you must first acquire a legal copy of a game that you wish to run, and then you can run a back-up copy of the game in question on commodity PC hardware.

MAME is also legal to use for "educational purposes" such as if you are learning how to construct emulation software.

Common Problems

  • Game Plays but Screen is Black - this is a problem with older OpenGL desktops. Switch video setting to "soft" (instead of OpenGL) in the configuration file.

MAME Friendly Hardware

Coin Slots

  • Happ Controls - This firm sells new parts for repairing old game machines, both video games and pinball. If you want to build a video arcade machine correct down to the working coin slot in the front you will want to look at this firm.

Joysticks

  • XGaming - Heavy duty, cabinet mountable, PC compatible joysticks.
  • Ultimarc - PC arcade game hardware, joysticks, trackballs and button to keyboard adaptors.
